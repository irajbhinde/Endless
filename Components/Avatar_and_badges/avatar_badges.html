Avatar
Avatars are user profile pictures, usually found in circular shapes. Avatars can be used for 'container' objects — projects, spaces, repositories, etcetera — to give them a recognisable visual identity.
Profile Avatar
Profile Avatars can be created using the class 'avatar', inside the img tag. 
        
    <-- Using the 'avatar' class name -->

    <img class="avatar" src="attach desired image" alt="avatar1">
    <img class="avatar" src="attach desired image">
Sizes Avatar
Size of the avatars can be changed by adding sm | md | lg class to the existing avatar class. 
        
    <-- Using the 'avatar' class name -->

    <img class="avatar sm" src="attach desired image">
    <img class="avatar md" src="attach desired image">
    <img class="avatar lg" src="attach desired image">
Square Avatar
We can build a square shaped avatar as well by adding 'square' class along with the desired size i.e sm | md | lg class. 
        
    <-- Using the 'avatar' class name -->

    <img class="avatar square sm" src="attach desired image">
    <img class="avatar square md" src="attach desired image">
    <img class="avatar square lg" src="attach desired image">
Badges
Badges are small circles, positioned either at top-right, top-left or bottom-right of the parent component. Badge can be used to display numbers, online / offline status, depending on where they are used.
Text with a Badge
        
    <-- Text with a badge -->

    <div class="bage_text" > <p>Text with a badge </p> /> 
    <div class="badge_icon" > <span> 2 </span> />

    <-- Text with a large badge -->

    <div class="bage_text" > <p>Text with a large badge </p> /> 
    <div class="badge_icon2" > <span> 2 </span> />
